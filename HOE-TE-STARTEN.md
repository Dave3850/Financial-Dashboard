# 🦷 CLINCH - ZO START JE DE APP (SUPER SIMPEL!)

## ✅ STAP 1: Installeer packages

Open de terminal in Cursor (zwart venster onderaan) en typ:

```bash
cd /workspace/server
npm install
```

Wacht tot klaar (30 seconden). Dan:

```bash
cd /workspace/client
npm install
```

Wacht tot klaar (1-2 minuten).

---

## ✅ STAP 2: Start backend

```bash
cd /workspace/server
npm start
```

Je ziet: `🦷 Clinch server draait op http://localhost:4000`

LAAT DIT VENSTER OPEN!

---

## ✅ STAP 3: Start frontend

Open een NIEUWE terminal (klik op + bij terminal onderaan) en typ:

```bash
cd /workspace/client
npm run dev
```

Je ziet: `Local: http://localhost:5173`

---

## ✅ STAP 4: Open in browser

Klik op de link `http://localhost:5173` OF open Chrome en ga naar:

```
localhost:5173
```

JE ZOU NU HET LOGIN SCHERM MOETEN ZIEN! 🎉
